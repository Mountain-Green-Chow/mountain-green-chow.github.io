农业文明面临工业文明的冲击

发表于 2019-05-04 更新于 2023-03-30 分类于 道理与逻辑 阅读次数： 本文字数： 3.3k 阅读时长 ≈ 3 分钟

不管我们愿不愿意承认，确实是鸦片战争的枪炮声，把我们从几千年的农耕文明迷梦中惊醒。当我们睁开双眼迷茫四顾时，才发现天朝上国万邦来朝的农业帝国，在工业机器面前那么不堪一击。如果不是两次鸦片战争，彻彻底底地把我们打疼了，我们还会沉醉在旧日的迷梦里不愿意醒来。大清帝国庞大的身躯，在几千年来以农耕文明为内在驱动力的强大惯性下，猛然强硬地向工业文明耦合，注定是不顺畅的。

自洋务运动开始，有识之士便大声疾呼我们要睁眼看世界，富国强兵，引进西方军事装备、机器生产和科学技术以挽救清朝的统治。

青山遮不住，毕竟东流去。

买了西方的武器机器，学习西方的科学技术，依然没能保住大清朝。每念及此，溥仪在劳改营无不痛心疾首地说：“俺的大清朝，亡了！”但是，向西方学习的脚步没有停下。

在发起洋务运动的时候，我们学习西方的指导思想就是：中学为体，西学为用！之后，进进退退，我们还是停留在这个思想水平上。不要认为自己坐在高楼大厦的办公室，吹着空调，刷着微信，就以为自己从肉体到灵魂，完成了工业文明现代化。真实情况是，往上数三代，不超过五代，我们都是刚刚洗脚上田的农民。西装革履包裹着的躯体里，是农耕文明的操作系统。对此，我们要有清醒的认识，不要自己迷惑自己。

对中体西用这个观点，周星驰曾经真诚地劝告过我们：那样只是得到我的肉体，并不能得到我的灵魂。老实巴交的农民，质朴中透漏着东方式的狡黠：“强扭的瓜虽然不甜，但是也能吃啊。灵不灵魂的无所谓，俺只要你的肉体，能生孩子就行！”

工业文明和农耕文明的内在有没有冲突，冲突在哪里？

我们是一个典型的农耕文明国家，1733年机械师凯伊发明了“飞梭”的时候，我们是雍正十一年，我们还在为康熙传位于四阿哥还是传位十四阿哥争论不休；1785年瓦特制成的改良型蒸汽机的投入使用的时候，我们是乾隆五十年，春：湖北宜城、光化、随州(今随县)、枝江大饥，人食树皮。秋：山东青州(今益都)府属寿光、昌乐、安丘、诸城等县大饥，父子相食。及至后来鸦片战争之前，东西方交流过程中，清朝对西方文明的傲慢和抗拒来看，我们接受工业文明都有难度，更何况孕育出工业文明。这说明工业文明不可能脱胎于农耕文明。

工业革命之前，除去农耕文明外，就是游牧文明和海洋文明了。工业文明如果不是从农耕文明中产生，就只能是在游牧文明和海洋文明中产生了。

农耕和渔猎是迥然不同的两种生产生活方式，生产生活方式不同，带来的思维方式也截然不同。农耕生产中，驯化的植物和动物是确定的，在某段时期甚至是固定的，土地也是固定的，生产方式有固定的时节顺序，每年的生产生活几乎是一成不变。农耕生产因为几乎没有什么增量，就只能在存量博弈中思考出路，钻研权谋之术，学习如何稳固自己和铲除异己，一切思考的出发点是人。渔猎生产则是另一番景象，每一种动物（包括鱼），生活习性都不同，那么针对不同种类的动物，渔猎方式都不同。而且随着时间的积累，遇到的动物种类也逐渐增加，对新动物习性的研究也要不断学习钻研。另一方面，如何增加捕获量，提高捕获效率，渔猎技巧也要不断改进，人与人的合作契合度也要不断增强。正因为渔猎生产方式，存量极其有限，而增量无限，所以只能在增量中思考出路，不断认知新生事物，发展新的渔猎技巧，一切思考的出发点是事和物。正是因为农耕和渔猎两种生产生活方式的不同，带来文明的走向也就不同了，日积月累，差异越来越大。、

到现在为止也是，我们世故老练有余，逢人必言做人，逢节必谈感恩，而思考创新不足，没有思想，少有发明。所谓做人和感恩，绝大多数是根据权力和位子来说的，对上谈感恩，对下谈做人，其实是一回事。

孔老二曾经曰过：君子和而不同，小人同而不和。就是说君子保持一团和气，但是各个心思不同；小人心思相同，却难保一团和气。把君子和小人改成农耕和渔猎，是不是非常契合？农耕生产里面，每个人都是可以替代的，所以：心里好气哦，但还是要保持微笑；最后结果就是只论亲疏，不论对错。没有对错，就没有真理，无法发展出逻辑和哲学，自然科学也并不在考虑之内。渔猎生产中，每个小团队，目标是一致的相同的，分工却各有不同，一个环节出错了带来的可能是整个团队的前功尽弃。所以，目标相同，错了，就不顾情面要追究责任了，也就是不和，要么改正错误，要么剔除出团队，否则整个团队就无法生存下去。所以，渔猎生产要严格区分对错，有对错就有逻辑，就能发展出有深度的思想，哲学和自然科学。渔猎的个体是有分工的，分工既会考虑个体特长的不同，也会发展出个体特长的不同，讲究合适的人放在合适的位置上，以保证效率最高产出最大。所以，渔猎生产是包容个体不同的。这一点和农耕生产截然不同。谁是君子，谁是小人？不同的生产方式，不同的文化背景，带来的结果是截然不同的。

美国总统特朗普开除前联邦调查局（FBI）局长詹姆斯·科米后，科米写了一本书《更高的忠诚：真相、谎言与领导力》，展开对特朗普的批评与攻击。特朗普也不甘示弱，在推特上对骂。这在西方司空见惯，要拿到中国来，那肯定是有失体统和不会做人。

对错只有公开才能让人分辨出对错，如果没有对错或者不谈对错，有何进步可言呢？

农耕和渔猎两种生产方式，在应对危机的时候，倾向性就完全不同。农耕几千年都是在存量里面打主意，每一次改朝换代，本质上不过是面对危机时社会财富的重新分配。每次社会财富的重新分配，带来的都是频繁反复的战争杀戮，百姓流离失所，生灵涂炭。所以我们一直有一种天下大同的理想，就是社会财富和资源永远是平均分配，这样就可以永保太平。渔猎的发展，则一直是寻找增量的过程，无论是发现新的物种，还是发明新的渔猎方法，亦或是发现新的财富资源，渔猎民族在寻找增量的路上，从未止步。所以，西方人的骨子里就有一股冒险精神和个人英雄主义。从海盗的打劫寻宝，到航海地理大发现，对每一种蝴蝶、每一种鸟、每一种鱼，都有细致入微的研究。这如果放在农耕民族里面，不是罪大恶极十恶不赦，就是不学无术不务正业，肯定得不到社会的认可。一个没有探索和创新的民族，谈何发展进步？

“中国的历史从本质上看是没有历史的，它只是君主覆灭的一再重复而已。任何进步都不可能从中产生。”出自黑格尔 《法哲学原理》。

农耕生产简单重复，依赖体力劳动，所以农耕民族的价值观导向是勤劳，所谓人勤地生宝，人懒地生草。渔猎生产讲究冒险和理性，谚语说：不敢冒险的人既无骡子又无马；过分冒险的人既丢骡子又丢马。这也反映了存量和增量思维下价值观念的不同。

农耕生产中，每一个个体都是一个独立完整的作业单元，劳动方式和劳动内容和其他人几乎一模一样，千人一面。所以，多你一个不多，少你一个不少。个人生存，小则依附于家庭宗族，大则依附于国家民族。个体的价值，在农耕社会里，无足轻重。正是这种千人一面的生产方式和依附于集体的生活方式，个人的个性、创新和爱好也是难以保持和发展的，一旦不见容于集体，个体的发展就趋于终点了。渔猎生产，每个个体的分工不同，需要优化组合每一个个人的特长，然后相互配合，才能最快捕获到最多的猎物，族群才能维系生存下去。因为存量有限，又需要每一个个体按照自己的爱好和特长，去探索新的领域，发现新的猎场，新的猎物或者更有效率的渔猎方法。单个个体，一旦开拓出一片新的领域，带来的增量是十分惊人的。西方经常用个人的名字命名一片土地、一个现象、一个定律、一个方程式或者一个星球，就是处于对个体价值的尊重。

美国的为什么叫America，它来自一个人的名字。在1500年代，意大利的商人和制图师 “Amerigo Vespucci“探索了南美洲的东海岸和加勒比海地区，最早绘制了新大陆的地图。他的名字用拉丁文写就是： Americus Vespucius。因为其他大陆用的名字都是女性化的拉丁语。所以，Americus 就变成了女性化的拉丁语： America。

正是因为渔猎民族的生存生活方式，要求他们尊重个体，勇于冒险开拓新的世界，不断学习新知识，不断尝试创新，才有了飞梭、织布机、蒸汽机、电力、电灯、手机、互联网和人工智能。

我们只学其形，不学其神，最终也是形似而神不似！

后发优势用尽了，后发劣势就会慢慢显现。
