农业文明的印记

发表于 2019-04-25 更新于 2023-03-30 分类于 道理与逻辑 阅读次数： 本文字数： 3.5k 阅读时长 ≈ 3 分钟

我们脚下这片土地在地理上有一万八千多公里的海岸线，之前我一直疑惑不解的是，为什么在之前的两三千年里，我们没能走向海洋文明，而走向了农耕文明？现在慢慢觉得，相比我们绵长的海岸线，我们内陆更加辽阔。在漫长的历史里，塑造我们这个民族性格和心理的，内陆一直处于绝对的支配地位，海洋对中国历史的影响，微乎其微——比如：我们没有一个皇帝来自渔民。

在工业革命发生以前，农耕相比渔牧是一个更加安稳有效率的模式。从生物学的角度来看，植物利用太阳能储存能量，动物以植物为食吸收植物储存的能量，人以动物为食吸取动物储存的能量，每个环节的能量损耗是惊人的。按人话来说就是：你吃两斤饭不能保证你长两斤肉，你吃两斤肉也不能保证你能长两斤肉，可能你吃吃喝喝一年了，最后一上称，你还掉膘了！所以，在工业革命以前，农耕模式在相同单位面积地表上，获取的能量最多，效能最大，更加有利于人类繁衍。

孙悟空同志曾经就说过：能看蟠桃园，谁还愿意去放马？为此，还闹过革命。

中国地域辽阔，有黄河、长江两大水系，东北、华北、关中和长江中下游四大平原，选择农耕是先民再合适不过的选择了。我们农耕的历史可以追溯到7000年前新石器时代的河姆渡文化，7000多年的农耕生活方式，已经将农耕的印记雕刻到我们的灵魂里了。

农耕的生产方式比较简单粗放，翻土，播种，施肥，除草除虫，收割，北方一年一季收成，中部一年两季收成，南部一年可达三季收成，年年如此。生产方式简单，带来的是人们思想的简单，发展不出有深度的思想，也就是哲学。说到这，比较有争议，一定有人要把孔子老子搬出来，我也只能装孙子了。

黑格尔说“我们看到孔子和他的弟子们的谈话（《论语》），里面所讲的是一种常识道德，这种常识道德我们在哪里都找得到，在哪一个民族里都找得到，可能还要好些，这是毫无出色之点的东西。孔子只是一个实际的世间智者，在他那里思辨的哲学是一点也没有了————只有一些善良的、老练的、道德的教训，从里面我们不能获得什么特殊的东西。西塞罗留下给我们的《政治义务论》便是一本道德教训的书，比孔子所有的书内容丰富，而且更好。我们根据他的原著可以断言：为了保持孔子的名声，假使他的书从来不曾有过翻译，那倒是更好的事”。（ 黑格尔《哲学史讲演录》第一卷）

这真的是黑格尔说的，和上面孙悟空同志说的不一样，而且我也是很认同黑格尔这一观点的。在1999年BBC评出的“世界千年十大思想家”里，也确实没有一个中国人。当然，也有很多榜单，孔子必居第一的，大家开心就好！

农耕的生产方式不但简单，而且脆弱。农业的育种、播种、栽种和收割，都是要顺应天时的，也就是后来总结出来的二十四节气，错过了一个环节就影响收成，造成减产减收。兔子这个月没打着，下个月还能打，稻子这个月没有种，下个月就晚了。另一方面，农作物在收割之前，毁坏掉是很容易的，而且很难补救。为了保证农业生产的顺利进行，人就会和相互信赖的人耕种生活在一起，通常是一个宗族的人，一个姓的人。所以，翻开现在的地图，很多农村的地名是跟姓相关的，比如周家大湾，王家湾，董家独屋，即使那个姓在那个地方早就不存在了。

农耕最重要的生产要素是土地，其次是水源。一方面农耕只能依赖于土地，土地是固定不可流动的，那么依赖于农耕生产的人也只能固化在这片土地上，人的活动半径是受限的，见识是有限的，更加不可能有深度的思想。人的活动半径受限就容易被奴役，财富难以保全，底层农民反而过的比渔牧民族更苦，遇上灾荒，卖儿卖女甚至吃人肉，在历史上一点不稀奇，甚至说是轻车熟路了。

土地有肥沃的，也有贫瘠的，水源有远近和充沛不充沛的问题。同时，农业又是个看天吃饭的行业，风调雨顺就还好，一旦遇到灾荒，个体的抗风险能力是非常有限的，必须要求助于家族亲戚，这就加强了宗族的凝聚力和号召力。宗族势力为什么能一直长期存在，就在于加强个体对天灾和人祸的抗风险能力。比如遇到干旱的年份，村里为了抢水吵架打架，村与村械斗，有过农村生活经历的人应该不会陌生吧，虽然现在没有宗族了，村里总会有人出面组织，有时候甚至就是村干部组织的，这时候村里武力值和霸气指数偏高的那一类人就派上大用场了，打赢了就回来大口吃肉大碗喝酒加吹牛——反正就互有输赢吧，总有赢的时候，没赢的都被饿死了！这是天灾，还有人祸——战争！广东人大致分为三大群体，先秦与百越融合的广府人、北方由海路南迁的潮汕人和北方由陆路南迁的客家人，为争夺土地，潮汕人、客家人均与广府人有过械斗，其中客家人与广府人，最长一次械斗持续了14年，死伤数十万人，背后的根源还是生存资源的争夺。

农耕文明的基础单位是宗族，不是家庭，不是个人。单个的个人和单个的家庭，在农耕社会里存活的概率实在太低了，几乎可以忽略不记。个人必须要依附于宗族才有存活下去的可能，既然如此，个人就必须要尽可能地抹除掉让别人不高兴的个体特征，尽可能让更多的人接受，个体才能获得尽可能多的利益，才能提高自我的生活质量。群体生活中，就必须要有长幼尊卑的秩序，必须人情往来给足面子，中国被尊为圣人的“孔子”的儒家思想，就是在这个基础上被广泛认同和发扬开来的。经年累月的宗族生活，也孕育诞生了中国式逻辑：问站队不问事实、问动机不问是非、问亲疏不问道理，至今还在或明或暗地延用。

土地有限，产出的资源也就有限，承载能力也是有限度的。人的繁衍能力是无限的，当土地不能承载繁衍带来的人口时，宗族的侵略性就被迫产生了，必须抢夺其他宗族的土地求生存。所以在中国以农耕为主导的历史上，宗族之间、姓之间、乡之间的械斗，从来没有停止过，多如牛毛！现在还能看到以前的祠堂，装饰琉璃瓦片和屋脊禽兽，极尽当时物力之奢华，也是一种无声的宣言——我宗族势力很大，你惹不起！宗族多以姓为标准，生儿子可以传宗接代，儿子多可以增强家庭势力、宗族势力，女儿武力值低又要嫁给外姓，中国至今很多地方仍然有根深蒂固的重男轻女思想。当然，联系外戚也可以增加势力，所以中国有一句古话叫多子多福。

农耕民族的武力值平均来看是很低的，和彪悍的游牧民族根本无法相提并论。所以，几千年来，除了官方组织，几乎没有农民自发地对渔牧民族发动过侵略战争，反而受渔牧民族的侵扰。中国北方是沙漠和深山老林，南边和东边是大海，西边是高原大山，都有不适宜耕种的地理隔离带。只要没地种，农民都懒得理你。但是只要有适于耕种的土地的地方，争斗就从来没有停止过。

在工业革命以前，我们还没有杜蕾斯，也没有学会节育手术，还没有电视机，天一黑，如果不喝酒不赌博，就只剩下造人这唯一的娱乐活动了。有限的土地与不断增长的人口的矛盾只能通过天灾或人祸来解决。天灾容易理解，影响有限，只是起到缩减人口的作用。人祸，通常就是小到宗族，大到国家的搏杀行为了。前面说过农耕生产有其脆弱性，配合宗族械斗，斗败的一方受到的几乎就是灭顶之灾了。胜利的一方顺着马太效应发展下去，入主中原，执掌天下就是顺理成章的了，之后就是呼皇喊帝称孤道寡了。再之后，建章立制，跟着以农耕生产方式为基础的一整套机构、法制、文化和社会组织结构就全面地建立起来，然后几千年地磨合运转下去，刻进我们的灵魂里。我们头脑里所思所想，至今都摆脱不了农耕文明的影响，我们生活里每一个细节，都刻有农耕文明的影子！

每一次改朝换代，大概率也有人口与土地矛盾的因素，地方矛盾频出，统治成本大增，当统治者的统治收益再也支付不起这统治成本的时候，就轰然倒塌皇朝改姓了。

以前看春秋战国的历史老是想不通，好端端的过日子不行吗，干嘛老是打来打去，其实就是因为农耕生产方式的产出有极限，无法匹配不断增长的人口。所以，必须要打，不是你死就是我亡！

想想这几千年来，我的老祖宗们都过得什么日子啊！

感谢工业革命，让人成其为人！

我们的肉身虽然迈进了工业文明，但我们的灵魂走得太慢，起码还有一半还卡在农耕文明里难以动弹！

在《哲学史讲演录》第一卷中黑格尔说：“意志有限性是东方人的性格，因为他们意志活动是被认作有限的，尚没有认识到意志的普遍性。在东方，只有主人与奴隶的关系，这是专制的阶段。在这个阶段里，恐惧一般地是主要的范畴。意志还没有从这种有限性里解放出来，因为思维本身也还不是自由的；因此意志可被认作是有限的，而有限的就可被假定为否定的。这种否定之感———感觉着某种东西不能长久支持下去———就是恐惧。反之自由却不是有限的，而是独立自在的，而独立自在的东西是不能被打倒的。人或是在恐惧中，或是有恐怖来统治人；二者是处在同一阶段。这一差别只在于一方有了较坚强有力的意志，它能够走向前去奴役一个切有限的意志使成为一个特殊的目的而牺牲。”

这是需要我们反思和检视的！
